Python Traffic Signal Simulator 🚦

Description

A basic Traffic Signal Simulator built using Python.

This project simulates a four-way traffic signal system with a countdown timer. The user can select between Normal Mode and Special Mode based on the time of day.

Features

- Four-way traffic signal simulation
- 30-second countdown timer
- Normal traffic signal operation
- Special mode for late-night hours
- User input validation
- Simple console-based interface

How It Works

Normal Mode

Available from 5:01 AM to 11:59 PM.

The signal cycles through:

1. Path 1 Green
2. Path 2 Green
3. Path 3 Green
4. Path 4 Green

Each path receives a 30-second countdown before the next signal activates.

Special Mode

Available from 12:00 AM to 5:00 AM.

All paths display Blinking Orange, indicating that drivers should proceed with caution.
